    bootstrap¶

    Provides the Beadledom bootstrap script along with an installer for the script.

    How to Install¶

    Running the following command will download and execute the beadledom install.sh script

    curl -L http://cerner.github.io/beadledom/4.1.2/beadledom-archetype/downloads/install.sh | sh

    Usage¶

    To see complete usage details use the following

    beadledom --help
