beadledom-resteasy-genericresponse
==================================

Overview
--------

This is an implementation of ``jaxrs-genericresponse`` for Resteasy. It should not be used directly, but instead simply included on the class path of Resteasy projects where it will be used automatically by ``jaxrs-genericresponse``.

Download
--------

Download using Maven:

.. code-block:: xml

  <dependencies>
      ...
      <dependency>
          <groupId>com.cerner.beadledom</groupId>
          <artifactId>beadledom-resteasy-genericresponse</artifactId>
          <version>[Insert latest version]</version>
      </dependency>
      ...
  </dependencies>
